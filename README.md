# Number Guessing Game

A simple Python game where the user tries to guess a randomly generated number between 1 and 100.

## Features
- Random number generation
- Higher/Lower hints
- Attempt counter
- User input handling

## Technologies Used
- Python
- Random Module

## How to Run
1. Clone the repository
2. Run:
   python guessing_no.py

## Sample Output
Enter a number: 50
Enter higher number

Enter a number: 75
Enter lower number

Correct answer!
You took 2 attempts to guess correctly.
